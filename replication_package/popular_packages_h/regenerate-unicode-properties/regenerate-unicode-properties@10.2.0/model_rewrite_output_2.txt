### Explanation
This Node.js code exports a `Map` object that categorizes Unicode properties. Each key in the `Map` is a string representing a category of Unicode properties, and each value is an array of strings, where each string is a specific property or script under that category. The available categories or keys in the `Map` are:

1. **General_Category**: Contains categories such as `Cased_Letter`, `Close_Punctuation`, etc., which classify characters into more general categories.
2. **Script**: Lists various language scripts like `Arabic`, `Cyrillic`, `Latin`, etc.
3. **Script_Extensions**: Similar to the `Script`, but includes scripts that may extend across multiple characters.
4. **Binary_Property**: Represents specific binary properties of characters such as `Alphabetic`, `Emoji`, `Uppercase`, etc.
