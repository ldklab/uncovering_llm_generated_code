The Node.js code exports a Map object containing different categories of Unicode properties. Each key in the map represents a category or property type, and the associated value is an array of strings representing the specific properties or subcategories within that type.

- `'General_Category'`: Includes subcategories such as 'Cased_Letter', 'Close_Punctuation', 'Connector_Punctuation', etc., which describe the broad classification of Unicode characters.
  
- `'Script'`: Contains various writing systems and scripts like 'Adlam', 'Ahom', 'Arabic', 'Armenian', etc., representing the script property of Unicode characters.
  
- `'Script_Extensions'`: Similar to 'Script', it includes scripts like 'Adlam', 'Ahom', 'Arabic', etc. This property provides additional script information that can be applied to characters.
  
- `'Binary_Property'`: Lists binary properties such as 'ASCII', 'Alphabetic', 'Any', 'Assigned', 'Case_Ignorable', etc., which provide additional metadata about the characters, typically in a true/false context.
