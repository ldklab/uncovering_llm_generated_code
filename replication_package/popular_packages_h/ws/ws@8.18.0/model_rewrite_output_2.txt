The functionality of the Node.js code provided is to set up and export a WebSocket utility package. It imports a core WebSocket module stored in `./lib/websocket` and enhances it with additional modules for handling WebSocket streams, servers, receiving messages, and sending messages. It then defines some aliases for these modules to make them easily accessible when the package is used elsewhere. Finally, it exports the WebSocket module with all enhancements.

Here's the rewritten code:

```javascript
'use strict';

const WebSocket = require('./lib/websocket');

// Enhance WebSocket with additional functionalities
WebSocket.createWebSocketStream = require('./lib/stream');  // Adds stream support
WebSocket.Server = require('./lib/websocket-server');       // Adds server capabilities
WebSocket.Receiver = require('./lib/receiver');             // Adds message receiving capabilities
WebSocket.Sender = require('./lib/sender');                 // Adds message sending capabilities

// Define module aliases
WebSocket.WebSocket = WebSocket;
WebSocket.WebSocketServer = WebSocket.Server;

// Export the enhanced WebSocket module
module.exports = WebSocket;
```
