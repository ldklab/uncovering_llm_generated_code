1. For Node.js 12 and above: The `process.allowedNodeEnvironmentFlags` Set includes flags that are valid for the current Node.js environment. The code checks if this Set contains the `--preserve-symlinks` flag, indicating support.

2. For Node.js versions between 6.2 and 11: It resorts to a more indirect method by inspecting the string of the internal `_findPath` method of `module.constructor`. If 'preserveSymlinks' is found in this string, it implies the flag's support.
