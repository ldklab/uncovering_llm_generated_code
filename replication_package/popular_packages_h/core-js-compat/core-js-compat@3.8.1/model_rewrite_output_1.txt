The functionality of this Node.js code is to create and export an object that combines various imported modules from different files. The Object.assign method is used to merge the imported 'compat' module with an object that contains all the imported modules as properties. The final result is exported so that it can be used in other parts of the application. The keys in the exported object correspond to the names of the modules, and the values are the imported modules themselves.

```javascript
'use strict';

const compat = require('./compat');
const data = require('./data');
const entries = require('./entries');
const getModulesListForTargetVersion = require('./get-modules-list-for-target-version');
const modules = require('./modules');

module.exports = {
  ...compat,
  compat,
  data,
  entries,
  getModulesListForTargetVersion,
  modules,
};
```
