The `cli-table3` Node.js package provides an API for creating text-based tables in the command-line interface. It builds upon two earlier libraries, `cli-table` and `cli-table2`, and offers features such as the ability to span columns and rows, customize cell styles, and handle different text alignments and word-wrap options. It is designed to be API-compatible with `cli-table` while providing additional functionalities and improvements, especially for text containing ANSI color codes.

Key features include:
- Customizable table characters and styles, including color and border styling.
- Column and row spanning, customizable column widths, and text truncation.
- Flexible text alignment and padding options.
- Built-in debugging to assist with table data visualization.

The library supports three types of tables:
1. **Horizontal Tables**: Tables where data is represented in rows, with a fixed header.
2. **Vertical Tables**: Tables where each item is represented by a key-value pair.
3. **Cross Tables**: A combination of horizontal and vertical tables where both headers and row labels are used.

Below is the implementation of a simplified version of the `cli-table3` package using Node.js:

```javascript
// cli-table3.js

class Table {
  constructor(options = {}) {
    this.options = options;
    this.rows = [];
  }

  push(...rows) {
    this.rows.push(...rows);
  }

  toString() {
    const { head = [], colWidths = [], chars = {}, style = {} } = this.options;
    const paddingLeft = style['padding-left'] || 1;
    const paddingRight = style['padding-right'] || 1;
    
    const defaultChars = {
      top: '-', 'top-mid': '+', 'top-left': '+', 'top-right': '+',
      bottom: '-', 'bottom-mid': '+', 'bottom-left': '+', 'bottom-right': '+',
      left: '|', 'left-mid': '|', mid: '-', 'mid-mid': '+',
      right: '|', 'right-mid': '|', middle: '|'
    };

    const c = { ...defaultChars, ...chars };
    const lines = [];

    const drawLine = (place) => head.length ? c[place + '-left'] + head.map((_, i) => c[place].repeat(colWidths[i])).join(c[place + '-mid']) + c[place + '-right'] : '';

    if (head.length) lines.push(drawLine('top'));

    if (head.length) {
      const header = head.map((h, i) => h.padEnd(colWidths[i] - paddingRight).padStart(colWidths[i] - paddingLeft)).join(c.middle);
      lines.push(c.left + header + c.right);
    }

    if (head.length) lines.push(drawLine('mid'));

    for (const row of this.rows) {
      const rowData = Array.isArray(row) 
        ? row.map((cell, i) => cell.toString().padEnd(colWidths[i] - paddingRight).padStart(colWidths[i] - paddingLeft)).join(c.middle)
        : Object.entries(row).map(([key, value]) => `${key}: ${value}`).join(' ');

      lines.push(c.left + rowData + c.right);
    }

    if (head.length) lines.push(drawLine('bottom'));

    return lines.join('\n');
  }
}

module.exports = Table;
```

This simplified implementation covers basic horizontal table creation with minimal options. It allows setting up custom headers, rows, and column widths. The table can be printed as a string using the `toString()` method.
