The provided Node.js code defines a module for escaping and unescaping HTML entities. It uses two main functions: `escape()` and `unescape()`. The `escape()` function replaces special characters (`&`, `<`, `>`, `'`, and `"`) in a string with their corresponding HTML entities to prevent HTML injection attacks. The `unescape()` function performs the reverse operation, converting HTML entities back to their original characters.

Here's a possible rewrite of the code:

```javascript
// html-escaper.js
const htmlEntities = {
  '&': '&amp;',
  '<': '&lt;',
  '>': '&gt;',
  "'": '&#39;',
  '"': '&quot;',
  '&amp;': '&',
  '&lt;': '<',
  '&gt;': '>',
  '&#39;': "'",
  '&quot;': '"'
};

function escape(str) {
  return str.replace(/[&<>'"]/g, char => htmlEntities[char]);
}

function unescape(str) {
  return str.replace(/&(amp|lt|gt|#39|quot);/g, entity => htmlEntities[entity]);
}

export { escape, unescape };
```
