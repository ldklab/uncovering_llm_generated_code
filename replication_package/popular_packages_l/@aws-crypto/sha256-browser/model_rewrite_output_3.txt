The provided Node.js code demonstrates how to compute a SHA-256 hash of a string using the `@aws-crypto/sha256-browser` library. The code performs the following steps:
1. Imports the `Sha256` class from the `@aws-crypto/sha256-browser` package.
2. Creates a new instance of the `Sha256` class, which is used to compute the hash.
3. Updates the hash instance with the string data 'some data'.
4. Calculates the final hash digest and awaits the result.

Below is a rewritten version of the code:

```javascript
import { Sha256 } from '@aws-crypto/sha256-browser';

async function computeHash(data) {
  const hash = new Sha256();
  hash.update(data);
  const result = await hash.digest();
  return result;
}

(async () => {
  const data = 'some data';
  const hashResult = await computeHash(data);
  console.log(hashResult);
})();
```
