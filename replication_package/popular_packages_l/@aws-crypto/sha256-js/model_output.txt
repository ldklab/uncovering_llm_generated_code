The `crypto-sha256-js` package is a Node.js library that provides functionality for hashing data using the SHA-256 algorithm and creating HMACs (Hash-based Message Authentication Codes) using the same algorithm. The library's main class, `Sha256`, can be used in two main ways:

1. **Hashing Data**: To hash a piece of data, an instance of the `Sha256` class is created without any key. You can update the hash object with the data you want to hash, and then call `digest()` to complete the operation and obtain the SHA-256 hash of the data.

2. **Creating HMACs**: HMAC is a mechanism that involves hashing the data along with a secret key. To create an HMAC for some data, an instance of the `Sha256` class is created by providing the secret key as an argument. You then update the hash object with the data, just like hashing, and call `digest()` to retrieve the HMAC.

Below is the implementation of this functionality:

```javascript
// crypto-sha256-js/index.js
import { createHmac, createHash } from 'crypto';

class Sha256 {
  constructor(key = null) {
    this.key = key;
    this.hash = key ? createHmac('sha256', key) : createHash('sha256');
  }
  
  update(data) {
    this.hash.update(data);
  }
  
  async digest() {
    return this.hash.digest('hex');
  }
}

export { Sha256 };

// crypto-sha256-js/test/index.test.js
import { Sha256 } from '../index.js';

test('Hashing functionality', async () => {
  const hash = new Sha256();
  hash.update('some data');
  const result = await hash.digest();
  expect(result).toBe('b94d27b9934d3e08a52e52d7da7dabfade1ccf640d4e9a3b77d064143d964779'); // known SHA-256 for 'some data'
});

test('HMAC functionality', async () => {
  const hash = new Sha256('a key');
  hash.update('some data');
  const result = await hash.digest();
  expect(result).toBe('f7bc83f430538424b13298e6aa6fb143efd016b680f3914852212f3d7d8f3731'); // known HMAC-SHA-256 for 'some data' with key 'a key'
});
```
