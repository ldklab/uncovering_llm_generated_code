The `requires-port` Node.js package is designed to determine if a specified port number is needed in a URL when using certain protocols. Given a port and a protocol, the function returns a boolean indicating whether the port is required in the URL.

## Installation
You can install the package using npm:
```
npm install --save requires-port
```

## Usage
The module exports a function that takes two arguments:
1. **Port**: This can be a number or a string representing the port number.
2. **Protocol**: A string representing the protocol. Supported protocols include `http`, `https`, `ws`, `wss`, `ftp`, `gopher`, and `file`. The protocol can be provided in various formats like `http`, `http:`, or even with a URL-like format such as `https://example.com`.

The function evaluates whether the specified port is required for the given protocol in a URL and returns `true` or `false`.

Example usage:
```js
var required = require('requires-port');

console.log(required('8080', 'http')) // true, because 'http' default is 80
console.log(required('80', 'http'))   // false, because '80' is the default HTTP port
```

Here is the implementation of the `requires-port` function:

```js
'use strict';

/**
 * Determine if the protocol requires a specified port.
 *
 * @param {String|Number} port - Port number to check.
 * @param {String} protocol - Protocol string to check against.
 * @returns {Boolean} - True if the port is required, false otherwise.
 */
function requiresPort(port, protocol) {
  // Convert port to a number
  port = Number(port);

  // Default ports for various protocols
  var protocols = {
    'http': 80,
    'https': 443,
    'ws': 80,
    'wss': 443,
    'ftp': 21,
    'gopher': 70,
    'file': null // file protocol doesn't use a port
  };

  // Extract the protocol name from input, ignoring any terminating slashes
  protocol = (protocol || '').split(':')[0].toLowerCase();

  // If the protocol is not recognized or doesn't typically use ports
  if (!protocols.hasOwnProperty(protocol) || protocols[protocol] === null) {
    return false;
  }

  // Return true if the provided port is not the default port for the protocol
  return protocols[protocol] !== port;
}

module.exports = requiresPort;
```
