The Node.js package `is-async-function` provides a utility to determine whether a given function is an asynchronous function, commonly known as an `async function`. Here's a summary of its functionality based on the provided README:

- **Purpose**: The package checks if a certain function is an `async function`. An `async function` is a native JavaScript function with the `async` keyword, which typically returns a Promise and allows for the use of `await` within it.

- **API**: 
  - It exports a single function `isAsyncFunction()`.
  - This function should take one argument (expected to be a function) and return a boolean:
    - `true` if the given function is an `async function`
    - `false` for any other kind of function or non-function input (including `null`, objects, or generator functions).

- **Example Usage**:
  - When given a regular function, a generator function, or `null`, `isAsyncFunction` should return `false`.
  - When given an `async function`, it should return `true`.

- **Testing your setup**: Users can run tests by cloning the repository, installing dependencies with `npm install`, and executing `npm test`.

Now, here's the implementation for the described functionality:

```javascript
// is-async-function.js
function isAsyncFunction(fn) {
    // Check if the input is a function and has a special constructor
    return typeof fn === 'function' && fn.constructor.name === 'AsyncFunction';
}

module.exports = isAsyncFunction;

// test.js
const assert = require('assert');
const isAsyncFunction = require('./is-async-function');

// Test cases
assert(!isAsyncFunction(function () {}), 'Regular functions should return false');
assert(!isAsyncFunction(null), 'Null should return false');
assert(!isAsyncFunction(function* () { yield 42; return Infinity; }), 'Generator functions should return false');
assert(isAsyncFunction(async function () {}), 'Async functions should return true');
```
