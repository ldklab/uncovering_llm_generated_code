```json
// package.json
{
  "name": "material-ui-custom-manager",
  "version": "1.0.0",
  "description": "Custom manager for handling Material UI installation and information",
  "main": "index.js",
  "scripts": {
    "install-material-ui": "node install-material-ui.js",
    "open-docs": "node open-docs.js"
  },
  "author": "Your Name",
  "license": "MIT",
  "dependencies": {
    "open": "^8.4.0"
  }
}
```

```javascript
// install-material-ui.js
const { exec } = require('child_process');

exec('npm install @mui/material @emotion/react @emotion/styled', (error, stdout, stderr) => {
    if (error) {
        console.error(`Error installing Material UI: ${error.message}`);
        return;
    }
    if (stderr) {
        console.error(`Installation stderr: ${stderr}`);
        return;
    }
    console.log(`Installation stdout: ${stdout}`);
});
```

```javascript
// open-docs.js
const open = require('open');

open('https://mui.com/material-ui/')
    .then(() => {
        console.log('Material UI documentation opened in browser.');
    })
    .catch((error) => {
        console.error(`Error opening documentation: ${error.message}`);
    });
```

The Node.js project consists of a script to handle Material UI installation and another to open its documentation. The `package.json` file registers commands `install-material-ui` and `open-docs`. When executed, `install-material-ui.js` installs Material UI packages using `npm`, and `open-docs.js` launches the default web browser to display the Material UI documentation page using the `open` package. Each script provides console feedback on success or error occurrences.
