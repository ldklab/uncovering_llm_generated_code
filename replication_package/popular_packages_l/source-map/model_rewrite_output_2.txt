   - Stores mappings and source contents.
   - Provides methods to add mappings, set source contents, and serialize these details to a source map format.

2. **SourceMapConsumer**:
   - Consumes an existing raw source map to interpret and provide original positions (line, column) based on generated positions in a compiled file and vice-versa.
   - Supports asynchronous consumption and cleanup of resources.
   - Includes utility methods for iterating over mappings.

3. **SourceNode**:
   - Models a tree of source content and maps chunks of it to their original positions, enabling the generation of both concatenated outputs and associated source maps.

        child.walk(fn);

    });
