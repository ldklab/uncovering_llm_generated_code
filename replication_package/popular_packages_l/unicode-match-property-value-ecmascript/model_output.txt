The `unicode-match-property-value-ecmascript` package provides a function to match a given Unicode property value or its alias to its canonical form, without applying loose matching, adhering strictly to ECMAScript's rules for Regular Expression Unicode property escapes. 

### Functionality Explained

1. **Purpose**: 
   - This function attempts to map a provided Unicode property value (or its alias) to its canonical property value for a given property.

2. **Input**:
   - A `property`: The canonical or unaliased name of a Unicode property.
   - A `value`: The value that needs to be matched against the canonical values for the given property.

3. **Output**:
   - If the `value` matches with a canonical Unicode property value, it returns this value.
   - If not, it throws an exception.

4. **Strict Matching**: 
   - The function enforces strict matching for the casing and naming conventions of property and value names, unlike "loose matching" which could allow variations like different cases.

### Code Implementation

Here's how you can implement this functionality in a Node.js package:

```markdown
// index.js
const propertyValueAliases = require('./unicode-property-value-aliases');

function matchPropertyValue(property, value) {
  if (!propertyValueAliases.hasOwnProperty(property)) {
    throw new Error(`Unknown property: ${property}`);
  }
  
  const canonicalValue = propertyValueAliases[property][value];
  
  if (!canonicalValue) {
    throw new Error(`Unknown value: ${value} for property: ${property}`);
  }
  
  return canonicalValue;
}

module.exports = matchPropertyValue;

// unicode-property-value-aliases.js

    'Caucasian_Albanian': 'Caucasian_Albanian',

  },
  // Additional properties and their potential values can be listed here
};

// Usage Example
const matchPropertyValue = require('./index');

try {
  console.log(matchPropertyValue('Script_Extensions', 'Aghb')); // → 'Caucasian_Albanian'
  console.log(matchPropertyValue('Script_Extensions', 'Caucasian_Albanian')); // → 'Caucasian_Albanian'
  console.log(matchPropertyValue('script_extensions', 'Caucasian_Albanian')); // → throws
} catch (error) {
  console.error(error.message);
}
```
