The `normalize-path` package is a utility for normalizing file paths in Node.js. Its main functionalities include:

1. **Forward Slash Conversion:** It converts backslashes (`\`) in paths to UNIX-like forward slashes (`/`). This makes paths consistent across different operating systems, such as Windows and UNIX-based systems.

2. **Condensing Repeated Slashes:** It reduces consecutive slashes in a path to a single slash. This ensures paths are clean and free of unnecessary redundancy, except for specific cases like leading slashes in Windows namespace identifiers.

3. **Trailing Slash Removal:** By default, any trailing slash at the end of a path is removed. However, the user can opt to retain the trailing slash by passing `false` as the second argument to the function.

Examples from the README demonstrate its usage:
- Normalizing a simple Windows path: `'\\foo\\bar\\baz\\'` becomes `'/foo/bar/baz'`.
- Handling Windows namespace paths: `'\\\\?\\UNC\\Server01\\user\\docs\\Letter.txt'` remains `//?/UNC/Server01/user/docs/Letter.txt`.
- Condensing extra slashes: `'.//foo//bar///////baz/'` becomes `'./foo/bar/baz'`.
- Keeping trailing slash when specified: `'foo\\bar\\baz\\'` becomes `'foo/bar/baz/'` when `false` is passed as second argument.

Here is the code implementation for `normalize-path` functionality:

```js
function normalizePath(path, stripTrailing = true) {
  if (typeof path !== 'string') {
    throw new TypeError('Expected a string');
  }

  // Check for Windows namespace
  const isWinNamespace = path.startsWith('\\\\.\\') || path.startsWith('\\\\?\\');
  
  if (isWinNamespace) {
    // Normalize starting double backslashes for win32 namespaces
    path = path.replace(/\\/g, '/');
  } else {
    // Replace backslashes with forward slashes
    path = path.replace(/[\\/]+/g, '/'); // condense slashes
  }

  // Remove trailing slash if stripTrailing is not explicitly set to false
  if (stripTrailing) {
    path = path.replace(/\/$/, '');
  }

  return path;
}

module.exports = normalizePath;

// Usage example
const normalize = require('./normalize-path');

console.log(normalize('\\foo\\bar\\baz\\')); 
//=> '/foo/bar/baz'

console.log(normalize('\\\\?\\UNC\\Server01\\user\\docs\\Letter.txt')); 
//=> '//?/UNC/Server01/user/docs/Letter.txt'

console.log(normalize('.//foo//bar///////baz/')); 
//=> './foo/bar/baz'

console.log(normalize('foo\\bar\\baz\\', false)); 
//=> 'foo/bar/baz/'
```
