The provided Node.js code is a simple HTML page with an embedded JavaScript script utilizing Popper.js to create a tooltip that appears next to a button when the mouse hovers over the button. The tooltip disappears when the mouse leaves the button.

### Explanation
1. **HTML Structure**: It includes:
    - A `<button>` with the text "I'm a button".
    - A `<div>` configured to serve as a tooltip with the initial style set to `display: none;` to keep it hidden until later.

2. **CSS Styling**: There is a style block in the `<head>` for the tooltip, giving it a dark background, white text, rounded corners, padding, and positioning.

3. **JavaScript Logic**:
    - The JavaScript is executed when the document's content is fully loaded (`DOMContentLoaded` event).
    - Event listeners are added to the button for `mouseenter` and `mouseleave` events.
      - `mouseenter`: When the mouse enters the button, the `show` function is called. This function makes the tooltip visible and uses Popper.js to position the tooltip to the right of the button.
      - `mouseleave`: When the mouse leaves the button, the `hide` function hides the tooltip and destroys the `popperInstance` to clean up resources.

4. **Popper.js Library**: The Popper.js library is included via a CDN to manage the positioning of the tooltip dynamically.

### Node.js Code Rewrite

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Popper Demo</title>
  <style>
    #tooltip {
      background-color: #333;
      color: white;
      padding: 5px 10px;
      border-radius: 4px;
      font-size: 13px;
      position: absolute;
      z-index: 1000;
      max-width: 200px;
    }
  </style>
</head>
<body>

  <button id="button" aria-describedby="tooltip">I'm a button</button>
  <div id="tooltip" role="tooltip" style="display:none;">I'm a tooltip</div>

  <script src="https://unpkg.com/@popperjs/core@2.11.6/dist/umd/popper.min.js"></script>
  <script>
    document.addEventListener('DOMContentLoaded', () => {
      const button = document.getElementById('button');
      const tooltip = document.getElementById('tooltip');

      let popperInstance = null;

      const createPopperInstance = () => {
        popperInstance = Popper.createPopper(button, tooltip, {
          placement: 'right',
        });
      };

      const showTooltip = () => {
        tooltip.style.display = 'block';
        createPopperInstance();
      };

      const hideTooltip = () => {
        tooltip.style.display = 'none';
        if (popperInstance) {
          popperInstance.destroy();
          popperInstance = null;
        }
      };

      button.addEventListener('mouseenter', showTooltip);
      button.addEventListener('mouseleave', hideTooltip);
    });
  </script>

</body>
</html>
```
