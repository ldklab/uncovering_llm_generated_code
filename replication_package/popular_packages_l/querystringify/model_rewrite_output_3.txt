The Node.js code defines a class `QueryStringify` that provides two static methods: `parse` and `stringify`. 

- **parse(query)**: This method takes a query string as input, removes leading `?` or `#` characters, and splits it into key-value pairs. It then constructs and returns an object where each key-value pair from the query string is represented as a property of the object. Key and value are URL-decoded.

- **stringify(obj, prefix)**: This method takes an object and optionally a prefix. It converts the object into a query string by encoding the object's keys and values as URI components and joining them with `&`. If a prefix is specified and is `true`, it prefixes the resulting string with a `?`. If the prefix is any other string, it prefixes the resulting string with that specific string. Without a prefix, it simply returns the constructed query string.

Here is the rewritten code:

```javascript
'use strict';

class QueryStringify {
  /**
   * Parse a query string into an object.
   * @param {String} query - The query string to parse.
   * @returns {Object} - The resulting object with key-value pairs.
   */
  static parse(query) {
    const object = {};
    query = query.replace(/^[?#]/, '');
    const pairs = query.split('&');
    for (const pair of pairs) {
      const [key, value = ''] = pair.split('=');
      if (key) object[decodeURIComponent(key)] = decodeURIComponent(value);
    }
    return object;
  }

  /**
   * Stringify an object into a query string.
   * @param {Object} obj - The object to stringify.
   * @param {Boolean|String} [prefix] - Boolean or string to prefix the query.
   * @returns {String} - The resulting query string.
   */
  static stringify(obj, prefix) {
    const str = Object.entries(obj)
      .map(([key, value]) => `${encodeURIComponent(key)}=${encodeURIComponent(value)}`)
      .join('&');

    if (!prefix) return str;
    return `${prefix === true ? '?' : prefix}${str}`;
  }
}

module.exports = QueryStringify;
```
