The Node.js code checks if the program is running on a Continuous Integration (CI) server using the `ci-info` package. It does this by checking environment variables specific to various CI vendors. If the program is running on a CI server, it logs the name of the CI server. If not, it logs a message indicating that the program is not running on a CI server. The `ci-info` package scrapes through known environment variables set by different CI systems to determine the server's presence and identity.

const ciInfo = {

  name: null,
  isPR: null

if (ciInfo.isCI) {
