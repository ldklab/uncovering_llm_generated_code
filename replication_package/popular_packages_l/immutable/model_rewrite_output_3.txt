The Node.js code is a basic implementation of immutable data structures, namely `ImmutableMap` and `ImmutableList`.

### Functionality Explanation:
1. **ImmutableMap**: This class is a simplified version of a map that maintains immutability.
   - **Constructor**: Initializes a map with the given object or defaults to an empty object.
   - **get**: Retrieves the value associated with a given key.
   - **set**: Returns a new instance of `ImmutableMap` with the specified key set to the new value. If the value is unchanged, it returns the same instance for optimization.
   - **equals**: Checks if two `ImmutableMap` instances have the same key-value pairs.

2. **ImmutableList**: This class is an immutable list similar to an array but doesn't allow direct modification.
   - **Constructor**: Initializes the list with the provided array or defaults to an empty array.
   - **get**: Accesses the element at the specified index.
   - **push**: Appends elements to the list and returns a new list instance.
   - **unshift**: Prepends elements to the list and returns a new list instance.
   - **concat**: Concatenates multiple `ImmutableList` instances and returns a new instance containing elements of all input lists.

```javascript
class ImmutableMap {
  constructor(initialObject = {}) {
    this._data = initialObject;
  }

  get(key) {
    return this._data[key];
  }

  set(key, value) {
    if (this._data[key] === value) {
      return this;
    }
    return new ImmutableMap({ ...this._data, [key]: value });
  }

  equals(otherMap) {
    const keys1 = Object.keys(this._data);
    const keys2 = Object.keys(otherMap._data);
    if (keys1.length !== keys2.length) {
      return false;
    }
    return keys1.every(key => this._data[key] === otherMap._data[key]);
  }
}

class ImmutableList {
  constructor(initialArray = []) {
    this._data = initialArray;
  }

  get(index) {
    return this._data[index];
  }

  push(...values) {
    return new ImmutableList([...this._data, ...values]);
  }

  unshift(...values) {
    return new ImmutableList([...values, ...this._data]);
  }

  concat(...lists) {
    return new ImmutableList(this._data.concat(...lists.map(list => list._data)));
  }
}

// Example usage:
const map1 = new ImmutableMap({ a: 1, b: 2, c: 3 });
const map2 = map1.set('b', 50);
console.log(map1.get('b')); // 2
console.log(map2.get('b')); // 50

const list1 = new ImmutableList([1, 2, 3]);
const list2 = list1.push(4, 5, 6);
console.log(list2.get(3)); // 4

module.exports = {
  ImmutableMap,
  ImmutableList,
};
```
