The code is a Node.js module that exports a function named `urlJoin`. This function concatenates multiple URL parts into a well-formed URL string. It takes an arbitrary number of string arguments which represent parts of a URL. The function flattens these parts, removes any empty strings, and joins them with slashes to form a complete URL. It also handles specific standard issues, ensuring no repeated slashes after the protocol, and makes sure that any query string starts with a single question mark.

Here's the rewritten Node.js code:

```javascript
// url-join.js

function urlJoin(...parts) {
  return parts
    .flatMap(part => part.split('/'))
    .filter(part => part !== '')
    .join('/')
    .replace(/(https?:\/)(\/+)/g, '$1/')
    .replace(/([^:])(\?{2,})/g, '$1?');
}

export default urlJoin;

// Example usage
const fullUrl = urlJoin('http://www.google.com', 'a', '/b/cd', '?foo=123');
console.log(fullUrl);
// Expected output: 'http://www.google.com/a/b/cd?foo=123'
```
