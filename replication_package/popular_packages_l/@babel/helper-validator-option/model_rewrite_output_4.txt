The Node.js code defines a class `OptionValidator` that validates a set of options against a predefined schema. The schema specifies rules for each option, including whether it's required, its type, and any allowed values. The `validate` method checks each option against these rules, collecting any validation errors. The `validate` method then returns an object indicating whether the validation passed and any error messages. An example usage demonstrates setting up a schema and validating a set of user-provided options.

Here is a possible rewrite of the code:

```javascript
// lib/index.js

class OptionValidator {
  constructor(schema) {
    this.schema = schema;
  }

  validate(options) {
    const errors = [];

    for (const [key, { required, type, allowedValues }] of Object.entries(this.schema)) {
      const option = options[key];

      // Required check
      if (required && (option === undefined || option === null)) {
        errors.push(`Option '${key}' is required.`);
        continue;
      }

      // Skip undefined non-required options
      if (option === undefined) continue;

      // Type check
      if (type && typeof option !== type) {
        errors.push(`Option '${key}' must be of type ${type}.`);
      }

      // Allowed values check
      if (allowedValues && !allowedValues.includes(option)) {
        errors.push(`Option '${key}' must be one of: ${allowedValues.join(', ')}.`);
      }
    }

    return {
      isValid: errors.length === 0,
      errors
    };
  }
}

// Example usage
const schema = {
  debug: { type: 'boolean', required: false },
  env: { type: 'string', required: true, allowedValues: ['development', 'production', 'test'] },
};

const validator = new OptionValidator(schema);

const userOptions = {
  debug: true,
  env: 'production'
};

const validationResult = validator.validate(userOptions);

if (!validationResult.isValid) {
  console.error("Validation failed with errors:", validationResult.errors);
} else {
  console.log("Validation succeeded");
}
```
