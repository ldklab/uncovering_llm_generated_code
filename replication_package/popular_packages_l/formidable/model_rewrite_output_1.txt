```javascript
import http from 'node:http';
import formidable from 'formidable';

const server = http.createServer((req, res) => {
  if (req.url === '/upload' && req.method.toLowerCase() === 'post') {
    const form = new formidable.IncomingForm();

    form.parse(req, (err, fields, files) => {
      if (err) {
        res.writeHead(400, { 'Content-Type': 'text/plain' });
        res.end('Error in file upload');
        return;
      }
      res.writeHead(200, { 'Content-Type': 'application/json' });
      res.end(JSON.stringify({ fields, files }, null, 2));
    });
  } else {
    res.writeHead(200, { 'Content-Type': 'text/html' });
    res.end(`
      <form action="/upload" method="post" enctype="multipart/form-data">
        <input type="text" name="title"><br>
        <input type="file" name="upload" multiple="multiple"><br>
        <button type="submit">Upload</button>
      </form>
    `);
  }
});

server.listen(8080, () => {
  console.log('Server listening on http://localhost:8080...');
});
```

**Explanation:** 
The Node.js code sets up a basic HTTP server using the `http` module. It listens for incoming requests on port 8080. When a request is received, it checks if the request is a `POST` to the URL `/upload`. For such requests, it uses the `formidable` module to parse any form data included in the request, which can include files. If parsing is successful, it responds with a JSON object containing both the fields and files posted by the client. If there's an error during parsing, it sends a plain-text error message. For all other request paths and methods, it responds with a simple HTML form, allowing the user to input a title and upload files using multipart/form data.
