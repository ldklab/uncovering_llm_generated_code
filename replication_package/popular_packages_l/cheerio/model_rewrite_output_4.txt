The Node.js code provided defines a simplified version of the jQuery-like library "Cheerio". This custom `Cheerio` class allows parsing and manipulation of HTML content with a basic set of functionalities: 

1. **Loading HTML Data**: Static method `load` wraps HTML content in a basic HTML structure and creates a `Cheerio` instance with a root element parsed from this structure.
   
2. **Finding Elements**: The `find` method uses CSS selectors to query elements from the parsed HTML.
   
3. **Extracting Text**: The `text` method retrieves and concatenates text contents of elements matched by the specified selector.
   
4. **Retrieving and Setting HTML**: The `html` method returns the HTML string of the parsed document. The `attr` method can retrieve an element's attribute or set an attribute's value for all matched elements.

Below is a rewritten version of the provided Node.js code encapsulating its functionality and logic:

```javascript
const { parse } = require('node-html-parser');

class SimpleCheerio {
  constructor(rootElement) {
    this.root = rootElement;
  }

  static load(htmlContent) {
    const root = parse(`<html><head></head><body>${htmlContent}</body></html>`);
    return new SimpleCheerio(root);
  }

  find(selector) {
    return this.root.querySelectorAll(selector);
  }

  text(selector) {
    const selectedElements = this.find(selector);
    return selectedElements.map(element => element.text).join('');
  }

  html() {
    return this.root.toString();
  }

  attr(selector, attribute, newValue) {
    const targetedElements = this.find(selector);
    if (newValue !== undefined) {
      targetedElements.forEach(element => element.setAttribute(attribute, newValue));
    } else if (targetedElements.length > 0) {
      return targetedElements[0].getAttribute(attribute);
    }
  }
}

// Example usage:
const cheerioInstance = SimpleCheerio.load('<ul id="fruits"><li class="apple">Apple</li><li class="orange">Orange</li></ul>');
console.log(cheerioInstance.html());

console.log(cheerioInstance.text('.apple'));

cheerioInstance.attr('li', 'class', 'fruit');
console.log(cheerioInstance.html());
```
