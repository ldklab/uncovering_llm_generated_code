The `available-typed-arrays` Node.js package is designed to determine which Typed Array types are available in the current JavaScript environment. Typed Arrays are a class of array-like objects that provide means to read and write raw binary data in memory buffers. The most common types include `Int8Array`, `Uint8Array`, `Uint8ClampedArray`, `Int16Array`, `Uint16Array`, `Int32Array`, `Uint32Array`, `Float32Array`, `Float64Array`, and the newer `BigInt64Array` and `BigUint64Array`.

The reusable function, `availableTypedArrays`, checks the current environment for the existence of these Typed Array constructors and returns an array containing the names of those that are available. This can be particularly useful in environments where support for certain Typed Arrays may be implementation-dependent or where feature detection is needed to ensure compatibility.

Below is the implementation of this package:

```markdown
// index.js
function availableTypedArrays() {
    const typedArrays = [
        'Int8Array',
        'Uint8Array',
        'Uint8ClampedArray',
        'Int16Array',
        'Uint16Array',
        'Int32Array',
        'Uint32Array',
        'Float32Array',
        'Float64Array',
        'BigInt64Array',
        'BigUint64Array'
    ];

    return typedArrays.filter(name => typeof global[name] === 'function');
}

module.exports = availableTypedArrays;

// test.js
const availableTypedArrays = require('./index');
const assert = require('assert');

const expected = [
    'Int8Array',
    'Uint8Array',
    'Uint8ClampedArray',
    'Int16Array',
    'Uint16Array',
    'Int32Array',
    'Uint32Array',
    'Float32Array',
    'Float64Array',
    'BigInt64Array',
    'BigUint64Array'
];

assert.deepStrictEqual(availableTypedArrays().sort(), expected.sort());
console.log('All tests passed!');
```

This package checks for the existence of Typed Array constructors within the `global` object in Node.js (or `window` in browser environments). The check `typeof global[name] === 'function'` determines if each Typed Array constructor is defined, thus indicating availability. It also includes a simple test in `test.js` to assert that all expected Typed Array names are correctly reported as available.
