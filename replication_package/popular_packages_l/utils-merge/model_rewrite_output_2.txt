The Node.js code is a simple utility module for merging two objects. The main functionality is provided by the `merge` function, which takes two arguments: `destination` and `source`. It ensures that both arguments are non-null objects and merges the properties from the `source` object into the `destination` object. If the keys exist in both objects, the `source` object's values overwrite the `destination` object's values for those keys.

Here's the rewrite of the Node.js code:

```js
// utils-merge.js
function mergeObjects(destObj, srcObj) {
  if (typeof destObj !== 'object' || destObj === null) {
    throw new Error('Destination must be a non-null object');
  }
  if (typeof srcObj !== 'object' || srcObj === null) {
    throw new Error('Source must be a non-null object');
  }
  
  for (let key of Object.keys(srcObj)) {
    destObj[key] = srcObj[key];
  }
  
  return destObj;
}

module.exports = mergeObjects;

// Usage example
const objA = { foo: 'bar' };
const objB = { bar: 'baz' };

console.log(mergeObjects(objA, objB));
// Output: { foo: 'bar', bar: 'baz' }
```
