The functionality of the given Node.js code is to provide a module with three validation functions: `isEmail`, `isAlpha`, and `isNumeric`.

1. `isEmail(str)`: Validates if the input string `str` is in the format of an email address. It uses a regular expression to match the common structure of an email: "text@text.text".

2. `isAlpha(str, locale, options)`: Checks if the input string `str` contains only alphabetic characters. The function supports different locales, though currently only 'en-US' is implemented, which includes characters from A-Z and a-z. An options parameter allows additional characters to be ignored using the `ignore` property, where you can pass a string of characters to include alongside the alphabetic check.

3. `isNumeric(str, options)`: Validates if the string `str` is numeric. It allows for an optional `options` parameter `no_symbols`, which when set to true, restricts the check to only digits with no symbols like plus, minus, or decimal points.

The example usage demonstrates calling the validation functions and checking their output, which returns true or false based on the validity of the provided input.

Here is the rewritten code:

```javascript
// validator.js

function isEmail(str) {
  const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return regex.test(str);
}

function isAlpha(str, locale = 'en-US', options = {}) {
  let regex;
  switch (locale) {
    case 'en-US':
      regex = /^[A-Za-z]+$/;
      break;
    default:
      regex = /^[A-Za-z]+$/; // Default to 'en-US'
  }

  if (options.ignore) {
    regex = new RegExp(`^[${regex.source}${options.ignore}]+$`);
  }

  return regex.test(str);
}

function isNumeric(str, options = { no_symbols: false }) {
  const regex = options.no_symbols ? /^[0-9]+$/ : /^[+-]?[0-9]+(\.[0-9]+)?$/;
  return regex.test(str);
}

// Example usage
console.log(isEmail('test@example.com')); // true
console.log(isAlpha('hello', 'en-US')); // true
console.log(isNumeric('123')); // true

// To use as a module in Node.js
module.exports = {
  isEmail,
  isAlpha,
  isNumeric
};
```
