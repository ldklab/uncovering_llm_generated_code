The given Node.js code consists of a simple Express server and a rudimentary client-side routing setup. The functionality can be explained as follows:

1. **index.js**: An Express server is created which listens on port 3000. The server has a single route ('/') that serves an HTML snippet to the client. This contains a:
   - `div` element with the ID 'root'.
   - A script tag to load `bundle.js`, which handles client-side rendering and routing.

2. **router-dom.js**: Implements a basic set of client-side routing components:
   - `BrowserRouter`: Encapsulates its children into a `div`. In a real-world scenario, this would handle browser history.
   - `Route`: Renders a component if the current path matches the specified path. It uses `window.location.pathname` to check the current URL and conditionally renders the component by invoking it.
   - `Link`: Creates an anchor link that, when clicked, changes the browser's URL without reloading the page using `window.history.pushState`.
   - `Switch`: Iterates over its children to render the first matching route. It stops iterating once it finds a route that should be displayed.

3. **bundle.js**: Contains the client-side rendering logic:
   - Defines two components, `Home` and `About`, returning simple strings representing the pages.
   - `App` defines the routes using `BrowserRouter`, `Link`, and `Switch`, and renders them into the root element.

```javascript
// index.js
const express = require('express');
const { BrowserRouter, Route, Link, Switch } = require('./router-dom');

const app = express();

app.get('/', (req, res) => {
  res.send(`
    <div id="root"></div>
    <script src="bundle.js"></script>
  `);
});

// Start Express server
app.listen(3000, () => {
  console.log('Server is running on http://localhost:3000');
});

// router-dom.js
module.exports = {
  BrowserRouter: function ({ children }) {
    return `<div>${children}</div>`;
  },
  Route: function ({ path, component: Component }) {
    const currentPath = window.location.pathname;
    return currentPath === path ? `<div>${Component()}</div>` : '';
  },
  Link: function ({ to, children }) {
    return `<a href="${to}" onclick="window.history.pushState({}, '', '${to}'); return false;">${children}</a>`;
  },
  Switch: function ({ children }) {
    for (const element of children) {
      if (element) return element;
    }
    return null;
  }
};

// bundle.js
const rootElement = document.getElementById('root');

function Home() {
  return 'Home Page';
}

function About() {
  return 'About Page';
}

function App() {
  return BrowserRouter({
    children: `
      ${Link({ to: '/', children: 'Home' })}
      ${Link({ to: '/about', children: 'About' })}
      ${Switch({
        children: [
          Route({ path: '/', component: Home }),
          Route({ path: '/about', component: About }),
        ]
      })}
    `
  });
}

rootElement.innerHTML = App();
```
